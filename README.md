# Лэндинг мобильного приложения FUN WEATHER
## В данном проекте был сделан лэндинг под ключ с использованием ванильных HTML/CSS/JS. Лэндинг полностью отзывчивый и имеет 3 брейкпоинта: под десктоп, таблет и мобильные устройства. Самым сложным в данном проекте была карусель виджетов, поскольку готовые решения не подходили под задачу и пришлось реализовать слайдер с нуля.

Условие выполнения: Сделать лэндинг под ключ с отзывчивой версткой. Из динамических элементов имеются только кнопки быстрого скролла вверх и вниз, а также карусель с имитацией 3D.

Решение: Для верстки были использованы HTML/SASS, JS библиотек использовано не было, поскольку ни одна не подошла под требования для реализации карусели. Во время создания карусели я столкнулся с необычной версткой (имитацией 3D).
